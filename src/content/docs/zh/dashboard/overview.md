# 仪表盘概览

> 适用版本：v1.0.0+
> 更新时间：2024-02-13

## 概述

Dashboard 是霖君的核心监控页面，提供实时的请求流量、Token 用量和成功率统计。

## 主要指标

### 请求统计

- **总请求数**：统计周期内的 API 请求总数
- **成功请求**：返回成功状态的请求数
- **失败请求**：返回错误或超时的请求数
- **成功率**：成功请求占总请求的百分比

### Token 用量

- **输入 Token**：发送给 AI 的 Token 数量
- **输出 Token**：AI 返回的 Token 数量
- **总 Token**：输入 + 输出的总和

### 响应时间

- **平均响应时间**：请求的平均耗时
- **P95 响应时间**：95% 请求的响应时间上限
- **最大响应时间**：最慢请求的耗时

## 提供商效率表

表格展示各提供商的详细统计：

| 字段 | 含义 |
|------|------|
| Provider | 提供商名称 |
| Requests | 请求总数 |
| Success Rate | 成功率百分比 |
| Avg Latency | 平均响应时间 |
| Tokens | Token 使用量 |

### Provider 名称显示规则

- **标准提供商**：显示服务名（如 `claude`、`gemini`）
- **自定义提供商**：显示为 `custom · 名称`
- **未识别**：极少数情况显示 `unknown`

## 时间范围

支持切换统计时间范围：

- 最近 1 小时
- 最近 24 小时
- 最近 7 天
- 最近 30 天

## 实时更新

Dashboard 会自动刷新数据，保持实时性。也可以点击刷新按钮手动更新。
