# 配额监控

> 适用版本：v1.0.0+
> 更新时间：2024-02-13

## 概述

Quota 页面用于监控各提供商账号的使用配额，帮助你了解账号状态和剩余用量。

## 主要功能

### 账号配额概览

每个已添加的账号会显示：

- **账号标识**：邮箱或账号 ID
- **提供商**：所属服务商
- **使用量**：当前周期已使用的配额
- **剩余配额**：可用的剩余额度
- **重置时间**：配额重置的时间点

### 配额预警

当账号配额接近用尽时，霖君会发出预警：

- **黄色警告**：配额使用超过 80%
- **红色警告**：配额使用超过 95%
- **通知**：可在设置中开启配额预警通知

## 查看所有模型

点击 **View All Models** 按钮可以查看提供商支持的模型列表：

- 模型列表来自 `/v1/models?is_webui=true` 接口
- 按提供商上下文过滤（如 Codex、Copilot、Kiro 各自显示对应模型）
- 显示模型 ID、名称和可用状态

## 提供商特定说明

### Claude

- 配额按月重置
- 显示剩余消息数或 Token 数

### Codex / Copilot

- 配额信息来自 GitHub 账号
- 可能显示订阅状态

### Gemini

- 显示 API 调用次数限制
- 按分钟/天/月分别统计

## 刷新配额

- 自动刷新：页面每隔一段时间自动更新
- 手动刷新：点击刷新按钮立即更新

## 常见问题

### 配额显示为空

- 可能是提供商未提供配额 API
- 尝试重新登录账号
- 检查网络连接

### 配额不更新

- 点击刷新按钮手动更新
- 部分提供商配额更新有延迟
