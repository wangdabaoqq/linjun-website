# 更新管理

> 适用版本：v1.0.0+
> 更新时间：2024-02-13

## 概述

霖君有两个组件需要更新：

- **应用本体**：霖君桌面应用
- **代理二进制**：CLIProxyAPIPlus 代理程序

## 检查更新

### 应用更新

1. 进入 **About** 页面
2. 在应用信息卡片中点击 **检查更新**
3. 如果有新版本，会显示更新按钮

### 代理二进制更新

1. 进入 **About** 页面
2. 在代理二进制卡片中点击 **检查更新**
3. 如果有新版本，点击 **更新** 下载

## 更新流程

### 更新应用本体

1. 点击更新按钮
2. 下载新版本安装包
3. 下载完成后提示重启
4. 重启后自动安装新版本

### 更新代理二进制

1. 点击更新按钮
2. 自动下载新版本二进制
3. 下载完成后自动替换
4. 代理服务自动重启

## 版本信息

在 About 页面可以查看：

- 当前应用版本
- 当前代理二进制版本
- 最后检查更新时间

## 自动更新

在设置中可以开启自动检查更新：

- 开启后，启动时自动检查更新
- 有新版本时显示通知
- 不会自动下载，需要手动确认

## 常见问题

### 更新下载失败

- 检查网络连接
- 尝试使用代理/VPN
- 稍后重试

### 代理二进制版本显示 unknown

- 可能是首次安装未下载
- 点击下载/更新按钮获取最新版本

### 更新后功能异常

- 尝试重启应用
- 检查 [更新日志](https://github.com/wangdabaoqq/LinJun/blob/main/CHANGELOG) 了解变更
- 如有问题，在 GitHub Issues 反馈
